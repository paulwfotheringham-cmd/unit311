## Outcome

Recruitment is a modern single-page ATS: KPIs, filters, vacancy management, Kanban board, candidate slide-over (Overview / CV / Interview Notes / Feedback / Offer / Timeline / Actions), create/edit modals, interview & offer recording, and bottom activity panels.

Demo-backed via the existing HR mock store. No additional navigation pages.
